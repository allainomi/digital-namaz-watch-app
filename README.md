## انسٹالیشن (Android / Wear OS)

1. **Unknown Sources** آن کریں (Settings > Security)  
2. `app-release.apk` ڈاؤن لوڈ کریں  
3. **Install** کریں  
4. **Location permission** دیں (نماز کے اوقات کے لیے)  
5. **کھولیں** → اذان سنئیے، قبلہ دیکھیے!
